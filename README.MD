# bfconsole
### Description
This is a brainf*ck emulator presented as a console.

### How to build
Install the following:
- [clang](https://clang.llvm.org) (or any C compiler, compiling with gcc looks like `make build CC=gcc`)
- [make](https://www.gnu.org/software/make)

Run `make`, this will list the supported recipes

### Implementation
- [x] Token parser
- [x] Token interpretation
- [ ] VRAM APIs
- [ ] Sprite loading
- [ ] Collision detection
